# DNS Server Lab

## Prerequisites
- Vagrant installed
- VirtualBox installed

## Quick Setup

```bash
vagrant up
```

## VM Configuration
- **OS**: DNS Server (bento/ubuntu-22.04)
- **Memory**: 4GB
- **CPUs**: 2 cores
- **Network**: Internal network (192.168.10.1)
- **GUI**: Enabled

## Connecting to the VM
```bash
vagrant ssh
```
